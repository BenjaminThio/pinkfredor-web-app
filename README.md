# Pinkfredor Web App
Online music player that streams music files directly from your google drive.

## Architecture
| Item     | Language |
| -------- | -------- |
| Frontend | Typescript React    |
| Backend  | NodeJS Express RESTful API |
| Storage  | Google Firebase Firestore |


## How to deploy
1. `npm install`
2. `npm run build`
3. `npm run deploy`
